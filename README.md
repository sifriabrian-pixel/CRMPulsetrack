# PulseTrack CRM Inmobiliario

CRM para gestión de leads desde Meta Lead Ads.

## Deploy en Vercel

1. Importar este repositorio en Vercel
2. Framework Preset: **Other**
3. Build Command: (vacío)
4. Output Directory: (vacío / raíz)
5. Click Deploy

## Funcionalidades
- Vista lista y Kanban de leads
- Dashboard con métricas y embudo
- Etapas del pipeline personalizables
- Notas por lead
- Exportar CSV
- Integración WhatsApp
- Google Calendar embebido (configurar en Ajustes)
- Webhook Meta Lead Ads (ver sección Webhook)
