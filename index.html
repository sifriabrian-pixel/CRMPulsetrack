Integración Meta Lead Ads
Conectá tus formularios para que los leads entren automáticamente
01 Datos que necesitás del cliente
1
Access Token de Meta
developers.facebook.com → Graph API Explorer. Permisos: leads_retrieval y pages_manage_metadata
2
ID de la Página de Facebook
Configuración → Información de la Página → ID
3
ID del Formulario de cada campaña
Meta Ads Manager → Formularios de leads → copiar ID
02 Cómo entran los leads (automático)
1
Cada cliente tiene su propia URL secreta
Sifer te la entrega al dar de alta al cliente. Cualquier sistema (Make.com, Meta Ads, o un bot de WhatsApp) puede mandar leads ahí — no hace falta login.
POST https://tudominio.com/api/webhook-lead/{tu-token}
2
Formato del lead
El mismo formato sirve para leads de Meta Ads y para leads calificados por un bot de WhatsApp.
{ "nombre": "Valentina", "email": "valentina@mail.com", "telefono": "+5493411234567", "propiedad": "Entre Ríos 1300", "respuesta_1": "...", "respuesta_2": "...", "respuesta_3": "..." }
03 WhatsApp — botón rápido
https://wa.me/{tel}?text=Hola+{nombre},+te+contactamos+por+{propiedad}
"Hola Valentina, te contactamos desde Sifer por tu consulta. ¿Tenés disponibilidad para coordinar una visita?"
Para automatización completa: WhatsApp Business API vía 360dialog o Wati
