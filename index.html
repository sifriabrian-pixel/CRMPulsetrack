Integración Meta Lead Ads
Conectá tus formularios para que los leads entren automáticamente
01 Datos que necesitás del cliente
1
Access Token de Meta
developers.facebook.com → Graph API Explorer. Permisos: leads_retrieval y pages_manage_metadata
2
ID de la Página de Facebook
Configuración → Información de la Página → ID
3
ID del Formulario de cada campaña
Meta Ads Manager → Formularios de leads → copiar ID
02 Configurar el Webhook
POST https://tudominio.com/api/webhook/meta-leads
1
Registrar en Meta for Developers
Tu App → Webhooks → suscripción al evento leadgen
2
Payload que llega
Meta envía el leadgen_id. Con ese ID se consulta la Graph API para obtener los datos completos.
{ "object": "page", "entry": [{ "changes": [{ "value": { "leadgen_id": "12345678", "form_id": "TU_FORM_ID" } }]}] }
03 WhatsApp — botón rápido
https://wa.me/{tel}?text=Hola+{nombre},+te+contactamos+por+{propiedad}
"Hola Valentina, te contactamos desde PulseTrack por tu consulta. ¿Tenés disponibilidad para coordinar una visita?"
Para automatización completa: WhatsApp Business API vía 360dialog o Wati

Etapas del pipeline
Renombrá las etapas del proceso de venta. Se aplican en la tabla, el kanban y los filtros. El orden importa: de izquierda a derecha representa el avance del deal.

Pegá el link de embed de tu Google Calendar para verlo dentro del CRM. Lo encontrás en Google Calendar → ⚙ → Configuración → tu calendario → "Integrar calendario" → "Código de inserción".
